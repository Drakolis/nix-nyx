# Product Overview

This is Mika Drakolis's personal NixOS system configuration repository. It provides a comprehensive, declarative configuration for multiple machines across different platforms (Linux/NixOS and macOS/Darwin) with consistent user environments.

## Key Features

- **Multi-platform support**: Supports both NixOS (Linux) and nix-darwin (macOS) systems
- **Multiple machine configurations**: Manages several named machines (Joermungandr, YDdraigGoch, Quetzalcoatl, WinterDragon, SilverWing, WorkerBee)
- **User-specific configurations**: Separate configurations for different users (drakolis, neinhorn, mikaz)
- **Desktop environments**: Supports multiple desktop environments including KDE Plasma, Hyprland, and Niri
- **Custom desktop shell**: Includes a custom Python-based desktop environment using Ignis
- **Development environment**: Comprehensive development tools and language support
- **Secrets management**: Uses SOPS for encrypted secrets management

## Target Users

Personal use configuration for the repository owner and family members across multiple devices and operating systems.
